# 籤詩文本研究與 pattern 分析（Phase 1）

> 「籤到」專案研究筆記 · 2026-07-15
> 資料來源：籤詩網（chance.org.tw）、台灣好廟網、TNL 關鍵評論網〈卦頭故事才是決定整首籤好壞的關鍵〉、
> 三鳳宮擲筊程序、威天宮線上抽籤、cecily.tw 求籤 SOP 等。

## 1. 台灣主要籤詩系統

| 系統 | 數量 | 常見廟宇 | 特徵 |
|---|---|---|---|
| 六十甲子籤 | 60 | 媽祖廟、天公廟 | 干支命名（甲子→癸亥），台灣最普及 |
| 雷雨師一百籤 | 100 | 關帝廟（行天宮等） | 數字命名，附「聖意」「東坡解」「碧仙註」 |
| 觀音一百籤 | 100 | 觀音寺（龍山寺等） | 上上/上/中/下/下下 等第明確 |
| 呂祖靈籤 / 月老籤 | 各異 | 仙公廟、月老殿 | 主題化（功名、姻緣） |

## 2. 一張籤詩的解剖（pattern）

台灣籤詩單張版面由以下元素構成（多寡依廟而異）：

1. **廟名／神明名**——權威來源（誰說的話）
2. **籤序**——干支或第 N 籤；六十甲子籤陽干配陽支、陰干配陰支
3. **吉凶等第**——大吉、上上、上吉、中吉、中平、下下……（各廟用語不一；中平佔比最高，大吉與下下皆稀少——「人生大多是中等日子」本身就是統計學上的誠實）
4. **卦頭故事（籤題）**——歷史／戲曲典故，如「蘇秦不第」「劉備招親」。
   TNL 的分析指出：**同樣詩句，卦頭故事才決定整首籤的走向**——它提供敘事框架，
   讓求籤者把自身處境投射到故事主人翁的處境上。
5. **四句詩文**——五言、七言或八言四句；意象多用自然節氣（雲開見日、枯木逢春、
   舟行遇風）隱喻運勢曲線；幾乎都描寫「狀態＋轉折」而非靜態結論
6. **解曰**——散文分項吉凶：功名、求財、婚姻、六甲、討海、行舟、病者、失物、
   出行、建居……（傳統農漁社會的需求清單）
7. **附註層**——聖意、東坡解、碧仙註、勸世警語（多層註解 = 不同「翻譯高度」）

### 詩文的修辭 pattern

- **氣象曲線**：前兩句鋪陳現況（常是困境或平淡），後兩句給出轉折與時機
  （「於今此景正當時」「雲開月出照天下」）——籤詩本質是**敘事弧**，不是判決書
- **時間提示**：常嵌入季節、月份（「不須作福不須求」「直到秋冬始安寧」）——
  給行動時機，而非抽象吉凶
- **條件句**：吉中帶誡（「若問前程去，猶如頂上鬃」）——保留能動性，
  解籤永遠給「所以你該怎麼做」
- **留白**：詩句刻意多義，由解籤人與求籤者共同完成意義——互動性是內建的

### 吉凶分布（重要設計參數）

六十甲子籤約：上籤 ~35%、中籤 ~45%、下籤 ~20%。大吉與下下皆為稀有事件，
抽中極端籤的「事件感」正來自稀有度。另有**籤王**（無字或特殊籤，機率極低）
作為系統的彩蛋層。

## 3. 求籤流程考據（互動設計的依據）

綜合三鳳宮、cecily.tw、TNL、寶島神很大之記載，完整 SOP：

1. **淨手上香**，向神明**稟告**：姓名、農曆生辰、住址、**所問之事**（一事一問，問題要具體）
2. **請示是否賜籤**——擲筊，**三聖筊**才可抽籤（有些廟一聖筊即可）
3. **搖籤筒抽籤**——取出跳出或抽出的那支籤
4. **擲筊確認「是否此籤」**——需**連續三聖筊**；笑筊＝再說清楚一次、陰筊＝不是這支，
   放回重抽
5. 取對應**籤詩紙**，依序排列
6. **解籤**——廟方解籤人依卦頭故事＋所問之事給出解讀

筊的三態：**聖筊**（一平一凸，允）、**笑筊**（雙平面朝上，神明笑而不答／問題不清）、
**陰筊**（雙凸朝上，否）。

### 對數位轉譯的關鍵洞察

- **儀式的核心不是隨機數，是「延遲與不確定」**：稟告→請示→抽→確認，每一步都可能
  被打回，這個「神明可以說不」的張力是紙本籤最迷人之處，也是大多數線上抽籤網站
  （直接按鈕→出結果）完全丟失的東西。威天宮線上抽籤已有先例，但互動深度有限。
- **默念所問之事**＝讓使用者先自我釐清問題——心理學上這一步本身就有價值
  （寫下問題 = 一半的解答）
- **抽籤是身體性的**：搖籤筒的聲音、籤支碰撞、擲筊落地脆響——手機上可用
  DeviceMotion（搖晃）、觸覺回饋（vibrate）、音效重建
- **解籤是對話**：解曰分項讓同一首籤回答不同的問題——線上版應讓使用者先選
  「問什麼」，再以對應解曰為主軸呈現

## 4. 給「籤到」的設計結論

1. 保留完整結構：籤序＋吉凶＋**卦頭故事（現代職場寓言）**＋七言四句＋解曰分項＋籤尾語
2. 吉凶分布仿傳統：大吉 8／上吉 20／中吉 36／中平 36／小凶 20／下下 8（共 128），
   極端籤保持稀有；另設籤王一支（機率 1/512）
3. 流程保留「請示→抽→**擲筊確認（可能被退籤）**」的張力，笑筊/陰筊都要出現
4. 解曰分項現代化：討海行舟六甲 → 事業、功名（升遷轉職面試）、財運、感情、健康、出行
5. 幽默放在卦頭故事與解曰，詩文保持古典尊嚴——反差即幽默
